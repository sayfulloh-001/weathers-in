# Weather App - Rain Edition ☔

Ob-havo ilovasi yomg'ir tomchilari va rangli animatsiyalar bilan.

## 🚀 Xususiyatlar

- Real vaqtda ob-havo ma'lumotlari
- 7 kunlik prognoz
- Yomg'ir tomchilari animatsiyasi
- Rangli interfeys (yashil, ko'k, qizil)
- Kecha/kunduz rejimi
- Xarita (sun'iy yo'ldosh, ko'cha, tungi)
- Harorat grafigi
- Havo sifati (AQI)
- Sevimlilar ro'yxati
- Qidiruv tarixi
- Ko'p tilli (UZ, RU, EN)

## 📦 O'rnatish

1. API kalitni oling: https://openweathermap.org/api
2. `.env` faylini yarating va kalitni qo'ying
3. `index.html` ni brauzerda oching

## 🔧 Sozlamalar

`.env` faylida quyidagilarni o'zgartirishingiz mumkin:
- API kalit
- Shahar
- Yomg'ir tomchilari soni
- Yangilanish oralig'i
- Boshqa sozlamalar

## 📱 Qo'llab-quvvatlanadigan qurilmalar

- Desktop
- Tablet
- Mobil telefon
